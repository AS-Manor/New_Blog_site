An Object is a complex data type that allows you to store key-value pairs. It provides a way to group related data and functions together. Here's how you can work with objects in JavaScript:


  1. Creating an Object:

    2. You can create an object by using curly braces { } and defining key-value pairs inside the object. Keys are strings, and values can be of any data type, including numbers, strings, booleans, arrays, or even other objects. Here are a few examples:

                    
var person = {
name: 'Abdul',
age: 21,
isStudent: true,
};

  2. Accessing Object Properties:

    3. You can access the properties of an object using dot notation(`objectName.propertyName`) or square bracket notation (`objectName['propertyName']`). Here are some examples;

                    
console.log(person.name); // Output: 'Abdul'
console.log(car['age']); // Output: '21'

  3. Modifying Object Properties:

    4. You can modify the value of an object's property by assigning a new value to it. Here's an example:

                    
person.age = 100;
person.isStudent = false;

  4. Adding New Properties:

    5. You can add new properties to an object by assigning a value to a new key. Here's some examples;

                    
person.gender = 'Male';
person.status = 'Single';
