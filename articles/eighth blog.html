In JavaScript, an array is a data structure that allows you to store multiple values in a single variable. Arrays can hold values of any data type, including numbers, strings, objects, or even other arrays. Here's how you can work with arrays in JavaScript:


  1. Creating and Array:

    2. You can create an array by using square brackets [ ] and separating the elements with commas. Here are a few examples:

                        
var numbers = [1, 2, 3, 4, 5]; // Array of numbers
var fruits = ['apple', 'banana', 'orange']; // Array of strings
var mixed = [1, 'hello', true]; // Array of mixed data types

  2. Accessing Elements:

    3. You can access individual elements in an array using their index. The index starts from 0 for the first element and increments by 1 for each subsequent element. Here's an example:

                        
var numbers = [1, 2, 3, 4, 5];
console.log(numbers[0]); // Output: 1
console.log(numbers[2]); // Output: 3

  3. Modifying Elements:

    4. You can modify elements in an array by assigning a new value to a specific index. Here is an example;

                        
var fruits = ['apple', 'banana', 'orange'];
fruits[1] = 'grape'; // Modifying the second element
console.log(fruits); // Output: ['apple', 'grape', 'orange']

  4. Array Length:

    5. You can determing the length of an array using the length property. It represents the number of elements in the array. Here is an example;

                        
var numbers = [1, 2, 3, 4, 5];
console.log(numbers.length); // Output: 5
