INTRO TO JavaScript

Author: Abdul Samed Manor
Date: 10th February 2023

JavaScript is a programming language that is well know for being the language of the web. However it has become so much more. Now, it is being used for both front-end development and back-end development. It works with HTML and CSS to create dynamic sites and apps.


You can use JavaScript in different ways;

  1. By adding it straight into HTML: Use the script tag and it is usually below the HTML body
  2. By creating a new .js file and linking it to the HTML file


Writing your first piece of JavaScript code


You can use VS code, Code Pen or using the Chrome browser iself - right click; click inspect; then click console.


To output a string in JavaScript; use

  • console.log ("Hello world");
  • document.write("Hello world");
  • alert("Hello world");


Variables in JavaScript


They are like containers for storing data. To declare variables, you can use; var, let, const. Below are some examples;

  • var x = 5
  • let name = "Abdul"
  • const rates = 0.5

NB: Numbers do not need to be enclosed in quotation. Only strings - series of characters.
