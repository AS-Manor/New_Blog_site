In JavaScript, loops allow you to execute a block of code repeatedly. The main types of loops are the for loop, while loop. Here's an overview of each:


  1. For Loop: The for loop is used when you know the number of iterations in advance. It consists of an initialization statement, a condition, and an increment or decrement statement. The loop will continue executing as long as the condition is true. Here's the syntax:
    2.                 
for (initialization; condition; increment/decrement) {
        // code to be executed
}
example:
for (var i = 0; i < 5; i++) {
    console.log(i);
  }
  2. While Loop: The while loop is used when the number of iterations is not known in advance and depends on a condition. It continues executing as long as the condition is true. Here's the syntax:
    3.                 
while (condition) {
    // code to be executed
}
example:
var i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
